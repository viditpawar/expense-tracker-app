# Expense Tracker App

A simple and modern expense tracking application built using Next.js and Tailwind CSS.

## Features

- Add expenses with title, amount, category, and date
- Filter expenses by category
- Delete individual expenses
- Clear all expenses
- Automatic total and filtered totals
- Data persistence using browser localStorage
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Vercel (deployment)
